# Contributing to SHITTS

We are building a high-performance, engineering-grade product. Contributions must meet this standard.

## Core Principles
1.  **Server is Authority**: No business logic in React components.
2.  **Immutability**: Financial records (Ledger) are forever.
3.  **Discipline**: Follow the domain models and API contracts.

## Process
1.  Fork the repo.
2.  Create a feature branch.
3.  Write tests for your changes.
4.  Ensure CI passes.
5.  Submit a pull request.
