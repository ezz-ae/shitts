# **App Name**: StyleSwipe

## Core Features:

- Product Swiping: Swipe right to add an item to the cart, left to reject it.
- Product Details: Swipe up for additional information about a product.
- Shopping Cart: Displays the list of selected clothing items with the ability to modify quantities or remove items.
- AI-Powered Style Recommendations: An AI tool analyzes the user's swiping history and purchase data to provide personalized style recommendations.

## Style Guidelines:

- Primary color: Vibrant pink (#FF69B4) to convey a fun and trendy shopping experience.
- Background color: Soft pastel pink (#FCE4EC), a desaturated version of the primary color, creates a gentle backdrop.
- Accent color: Purple (#A020F0) provides contrast for call-to-action buttons.
- Body and headline font: 'Poppins' sans-serif font, provides a contemporary, geometric style. Note: currently only Google Fonts are supported.
- Use minimalist and recognizable icons for actions (e.g., heart for 'like', cross for 'dislike', cart for 'add').
- Full-screen product images for immersive experience; swipe gestures for intuitive navigation.
- Smooth transitions during swiping; subtle animations for item adding to the cart.
